1) Project Description

What it is, why it exists, who it helps, how it works, why support it

What is NorthStar?
NorthStar is a web platform that helps people explore education and career pathways in a structured, easy way.

Problem it solves
Many students (and parents) feel overwhelmed by too many choices, unclear requirements, and confusing advice. This leads to stress, wasted time, and choosing a path without confidence.

Why the problem matters
Education and career decisions affect finances, motivation, and long-term opportunities. Getting direction early can reduce dropouts, improve planning, and increase satisfaction.

What makes it unique
NorthStar combines: simple explanations + curated resources + a step-by-step decision flow (not just “lists of professions”).

How it works (simple flow)

  1. Answer quick questions about interests, strengths, and goals.
  2. Get suggested pathways (majors, careers, short courses).
  3. See requirements, time, cost, and possible next steps.
  4. Save a plan and track progress with a timeline.

Why people should support it

  • Helps young people make informed decisions.
  • Reduces confusion and “random choices”.
  • Creates equal access to guidance and opportunities.

2) Functional Requirements

What the product must do

Core Features

  • User can explore education and career options by category.
  • User can take a short “pathfinder” quiz and get suggestions.
  • User can view each pathway with skills, time, and requirements.

User Accounts

  • User can register/login (optional MVP if time is limited).
  • User can save favorites (careers, majors, resources).
  • User can create a simple personal plan/checklist.

Resources & Support

  • User can access curated resources (links, scholarships, courses).
  • User can find opportunities (events, internships, competitions).
  • User can contact/support via feedback form.
